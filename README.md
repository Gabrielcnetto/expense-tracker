Manage your expenses with ease using Expense Tracker CLI


Complete code available at https://github.com/Gabrielcnetto/expense-tracker

Usage:
  github-user-activity [flags]


Use "Task [command] --help" for more information about a command.

## CLI Commands


```bash
Usage:
  expense-tracker [command]

Available Commands:
  add         Add new Expense to list
  completion  Generate the autocompletion script for the specified shell
  delete      Remove expense of list
  help        Help about any command
  list        Get all expenses added by you
  summary     Get a sum of all expenses that you added

Flags:
  -h, --help     help for expense-tracker
  -t, --toggle   Help message for toggle

Use "expense-tracker [command] --help" for more information about a command.

```
